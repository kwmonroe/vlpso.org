# Static Website for School Grades

This project is a static website that provides information and resources for school grades from Kindergarten through 12th grade. The website features a side navigation menu that allows users to easily navigate between different grade pages.

## Project Structure

```
static-website
├── src
│   ├── index.html          # Main entry point of the website
│   ├── styles              # Contains CSS styles
│   │   └── style.css       # Styles for the website
│   ├── scripts             # Contains JavaScript files
│   │   └── app.js          # JavaScript for dynamic behavior
│   ├── pages               # Contains individual grade pages
│   │   ├── kindergarten.html
│   │   ├── grade1.html
│   │   ├── grade2.html
│   │   ├── grade3.html
│   │   ├── grade4.html
│   │   ├── grade5.html
│   │   ├── grade6.html
│   │   ├── grade7.html
│   │   ├── grade8.html
│   │   ├── grade9.html
│   │   ├── grade10.html
│   │   ├── grade11.html
│   │   └── grade12.html
├── README.md               # Documentation for the project
└── assets                  # Contains images and fonts
    ├── images
    └── fonts
```

## Getting Started

To set up and run the website locally, follow these steps:

1. Clone the repository to your local machine.
2. Navigate to the `static-website` directory.
3. Open `src/index.html` in your web browser to view the website.

## Features

- Side navigation for easy access to each grade page.
- Responsive design for optimal viewing on various devices.
- Custom styles and scripts to enhance user experience.
